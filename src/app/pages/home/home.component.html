Learning skeleton

Build your own NgOptimizedImage examples one topic at a time.

This project provides routing, layout, notes, and verification prompts. The actual NgOptimizedImage code is intentionally left for you to add inside each exercise slot.

Suggested Workflow

  1. Open one demo route.
  2. Add only the image code needed for that concept.
  3. Run the app and inspect DOM, Network, Console, and Performance behavior.
  4. Document what changed before moving to the next route.
